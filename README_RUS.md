# MTP плагин для Double Commander

[In english](README.md)

Это плагин для файлового менеджера [Double Commander](https://doublecmd.sourceforge.io) в операционной системе macOS.

Плагин позволяет файловому менеджеру Double Commander взаимодействовать с файлами на устройствах передающих данные по протоколу MTP (Media Transfer Protocol).

## Как установить и настроить

### 1. Установить зависимости
Для работы с MTP устройствами плагин испльзует [libmtp](https://github.com/libmtp/libmtp). Поэтому требуется установить libmtp. Самый простой вариант: установить через [Homebrew](https://brew.sh).

```
brew install libmtp
```

### 2. Добавить плагин в Double Commander
1. Скачать плагин (файл *.wfx64) со страницы релизы (Releases) этого репозитория или получить этот файл, скомпилировав плагин из исходного кода.
2. В программе Double Commander открыть окно настроек: ***Настройки > Параметры*** 
3. В окне настроек в боковом меню выбрать пункт ***Плагины WFX***
4. Нажать кнопку ***Добавить*** и указать путь к файлу плагина (файл с расширением **.wfx64**).

## Как использовать
1. Подключить устройство MTP к компьютеру по USB, далее в настройках устройства включить передачу данных по протоколу MTP (в современных Android смартфонах режим *"Передача файлов/Android auto"*).
2. Открыть Double Commander, далее в Double Commander открыть вкладку **//** (**Открыть список VFS**). На вкладке **vfs:** открыть папку **MTP**. В этой папке будут отображаться подключенные устройства MTP.

## Для разработчиков

1. Для компиляции плагина нужен файл заголовка libmtp и установленная библиотека libmtp (версии 1.1.22). 

    Установка libmtp:
    ```
    brew install libmtp
    ```

2. Для компиляции плагина использовался компилятор clang++ входящий в состав Xcode Command Line Developer Tools. 

    Установка Xcode Command Line Developer Tools:
    ```
    xcode-select --install 
    ```

3. Для компиляции плагина: отметить скрипт компиляции как исполняемый файл и запустить.

    Для mac на процессоре Apple Silicon:
    ```
    chmod +x compile_mac.sh
    ./compile_mac.sh
    ```

    Для mac на процессоре Intel:
    ```
    chmod +x compile_mac_x86_64.sh
    ./compile_mac_x86_64.sh
    ```

    Скомпилированный плагин, файл **fsplugin.wfx64**, будет располагаться в той же папке.
